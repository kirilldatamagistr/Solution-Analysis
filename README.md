## 🚀 Проекты

### 1. 📞 Анализ call-центра (теория массового обслуживания)

**Задача:** Определить оптимальное число операторов и размер очереди для call-центра.

**Реализовано:**
- Три модели СМО: `M/M/N/0`, `M/M/N/K`, `M/M/N` с двумя неоднородными потоками
- Расчёт вероятности отказа, простоя, средней очереди
- Поиск минимального N и K при заданном уровне потерь

**Технологии:** `numpy`, `scipy.special`, `math`

---

### 2. 🖥️ Edge-Fog-Cloud симуляция

**Задача:** Смоделировать работу трёхуровневой вычислительной системы (периферия → туман → облако).

**Реализовано:**
- Simpy-модель с `Container` для каждого уровня
- Разделение потока задач с весами `[0.05, 0.15, 0.8]`
- Разная скорость обработки на каждом уровне
- Визуализация динамики загрузки и гистограммы распределения

**Технологии:** `simpy`, `random`, `matplotlib`, `numpy`

---

### 3. 📱 Телеком-оператор (аналитика использования услуг)

**Задача:** Проанализировать поведение клиентов на основе данных о звонках, сообщениях и интернет-трафике.

**Реализовано:**
- Предобработка: приведение типов, очистка, удаление выбросов
- Расчёт помесячной выручки с учётом тарифных лимитов
- Визуализация распределений (гистограммы)
- Проверка гипотез:
  - Различается ли выручка тарифов «Базовый» и «Смарт»?
  - Различается ли выручка Москвы и регионов?

**Технологии:** `pandas`, `matplotlib`, `seaborn`, `scipy.stats`

---

### 4. 🏋️ Фитнес-центр (прогнозирование оттока)

**Задача:** Построить модель прогнозирования оттока клиентов и сегментировать аудиторию.

**Реализовано:**
- EDA и визуализация факторов оттока (пол, контракт, локация, промо)
- Корреляционная матрица
- Модели: **Логистическая регрессия** и **Random Forest**
- Метрики: Accuracy, Precision, Recall
- Кластеризация: `KMeans` (4 кластера) + дендрограмма + PCA визуализация

**Технологии:** `sklearn`, `scipy.cluster.hierarchy`, `pandas`, `matplotlib`, `seaborn`

---

### 5. 🛒 Маркетплейс «ВезуМаркет» (сквозной анализ)

**Задача:** Провести полный анализ данных интернет-магазина и оценить эффективность внедрения чат-бота.

**Реализовано:**
- Предобработка и расчёт интенсивности потока (`λ`) по месяцам
- Анализ лояльности: распределение по количеству визитов, средний чек
- Статистические тесты:
  - Сравнение среднего чека (женщины vs мужчины) — **U-тест Манна–Уитни**
  - Сравнение доли консультаций по платформам — **Z-тест для пропорций**
- Имитационная модель чат-бота (SimPy):
  - Расчёт `λ` и `μ` из данных
  - Поиск оптимального лимита (цель ≤10% отказов)
  - Сравнение с формулой **Эрланга B**
- Сегментация клиентов (self-sufficient / needs support)
- **ROI-анализ** внедрения чат-бота:
  - Экономия на консультантах (40% автоматизация)
  - Дополнительная выручка (+15% конверсии)
  - Срок окупаемости, чистая прибыль, ROI

**Технологии:** `pandas`, `numpy`, `scipy.stats`, `statsmodels`, `simpy`, `matplotlib`, `seaborn`
